Estrutura Básica do HTML

(Voltar)

Uma estrutura básica de HTML é o esqueleto fundamental de uma página da web. Ela consiste em elementos essenciais que ajudam a organizar e exibir o conteúdo de forma adequada no navegador.
Aqui está um resumo dos principais componentes da estrutura básica de HTML:

  1. Tag < !DOCTYPE html >: Indica a versão do HTML utilizada na página.
  2. Tag < html >: Delimita o início e o fim do documento HTML.
  3. Tag < head >: Contém informações sobre o documento, como metadados, títulos, links para arquivos externos (CSS, JavaScript, ícones) e outros elementos que não são visíveis ao usuário.
  4. Tag < meta charset="UTF-8 >: Define o conjunto de caracteres utilizado no documento, geralmente definido como UTF-8 para suportar caracteres internacionais.
  5. Tag < title >: Define o título da página que será exibido na aba do navegador ou na janela do navegador.
  6. Tag < body >: Contém todo o conteúdo visível da página, como texto, imagens, links, elementos multimídia, formulários, etc.
  7. Outras tags: Dentro do < body >, você usará diversas outras tags HTML para estruturar e formatar o conteúdo, tais como:
    • Títulos: < h1 >, < h2 >, < h3 >, < h4 >, < h5 >, < h6 >
    • Parágrafos: < p >
    • Imagens: < img >
    • Links: < a >
    • Listas: < ul >, < ol >, < li >
    • Tabelas: < table >, < tr >, < td >
    • Formulários: < form >, < input >, < textarea >, etc.

Essa estrutura básica forma o alicerce de qualquer página web, permitindo que os navegadores interpretem e exibam o conteúdo corretamente aos usuários. Combinando esses elementos com CSS
(folhas de estilo) e JavaScript (linguagem de programação) é possível criar páginas web ricas e interativas.

O que são Tags?

(Voltar)

Tags em HTML (HyperText Markup Language) são elementos especiais utilizados para marcar e estruturar o conteúdo de uma página web. Elas fornecem instruções ao navegador sobre como exibir
o conteúdo e definir sua apresentação na página. As tags são compostas por palavras-chave envoltas por colchetes angulares ("<" e ">").

Cada tag pode ter um propósito específico e pode conter atributos que modificam o comportamento ou a aparência do elemento. As tags podem ser usadas para inserir texto, imagens, links, tabelas,
formulários e outros elementos que compõem a página.

As tags são fundamentais para a estruturação e a formatação correta de uma página web e permitem que os navegadores interpretem o conteúdo e o apresentem adequadamente aos usuários. O uso
correto e semântico das tags é essencial para criar páginas acessíveis e bem organizadas.

Fora as tags já vistas no capitulo anterior, existem também:

  • Tag < mark > é usada para destacar ou marcar partes do texto dentro de um parágrafo ou bloco de texto.
    Exemplo: Usando a tag < mark >.
  • Tag < small > é usada para definir um texto que deve ser apresentado em tamanho de fonte menor do que o texto circundante.
    Exemplo: Usando a tag < small >.
  • Tag < i > é usada para definir texto que deve ser exibido em itálico.
    Exemplo: Usando a tag < i >.
  • Tag < u > é usada para definir texto que deve ser sublinhado.
    Exemplo: Usando a tag < u >.
  • Tag < strong > é usada para definir texto que deve ser exibido em negrito.
    Exemplo: Usando a tag < strong >.

Entre várias outras tags que existem.

O que são os atributos das Tags?

(Voltar)

Os atributos são elementos adicionais usados em tags HTML para fornecer informações extras ou configurações específicas para essas tags. Eles são utilizados para modificar o comportamento
ou a aparência dos elementos, personalizando a forma como eles são exibidos na página ou fornecendo informações adicionais para processamento pelos navegadores ou outras tecnologias.

Existem muitos atributos diferentes em HTML, como href, class, style, target, disabled, placeholder, required, entre outros, e cada um deles tem uma finalidade específica de acordo com a tag
em que são usados. O uso adequado dos atributos é essencial para garantir a estruturação correta e a acessibilidade das páginas da web.

Listas Ordenadas e Não Ordenadas

(Voltar)

Listas ordenadas e listas não ordenadas são dois tipos de listas HTML que permitem agrupar e exibir itens relacionados em uma sequência específica ou não específica.

Uma lista ordenada (< ol >) é uma lista de itens que são organizados em uma sequência numérica ou alfabética. Cada item da lista é representado por uma tag < li > (item da lista) e, por padrão, é exibido
com um marcador numérico (1, 2, 3, etc.) ou alfabético (a, b, c, etc.) ao lado do texto do item.

Uma lista não ordenada (< ul >) é uma lista de itens que não segue uma sequência numérica ou alfabética específica. Cada item da lista também é representado por uma tag < li > e, por padrão, é exibido
com um marcador de ponto (•), mas a aparência pode variar dependendo do navegador ou das folhas de estilo (CSS) aplicadas.

Ambos os tipos de listas (< ol > e < ul >) são amplamente usados em páginas da web para organizar informações, criar menus, listas de itens, entre outras situações em que a apresentação em forma de
lista é adequada. O conteúdo dentro de cada < li > pode conter qualquer estrutura HTML desejada, permitindo a inclusão de imagens, links, parágrafos e outros elementos.

Links

(Voltar)

Links, também conhecidos como hyperlinks ou simplesmente "links", são elementos fundamentais em páginas da web que permitem a navegação entre diferentes recursos ou páginas na internet.
Eles permitem que os usuários cliquem em um texto ou em uma imagem para serem redirecionados para outra página, documento, imagem, vídeo, ou qualquer outro tipo de recurso online.

Os links são criados usando a tag < a > em HTML, que possui um atributo href que especifica o destino do link, ou seja, a URL do recurso que será acessado quando o link for clicado.

Os links são essenciais para a navegação na web, pois permitem que os usuários explorem e acessem diferentes páginas e recursos sem ter que digitar manualmente as URLs completas.
Eles também são usados para conectar diferentes páginas em um site, criar menus de navegação e fornecer acesso rápido a informações relevantes.
